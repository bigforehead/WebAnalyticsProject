**WebAnalyticsProject**
*This program is created to crawl basic information of Ice Cream shops from Yelp*
- Ranking
- Name
- Review
- Rating
- Phone
- Address
- District
Applied BeautifulSoup package to crawl 100 Ice Cream shops - Methodology: extract data from json file 
Applied pandas package to output the information into one table
Applied matplotlib to create charts of average rating and number of restaurant based on their location by district
